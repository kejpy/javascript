Pętla for

Konstrukcja:
for (inicjalizacja; warunek; zmiana) { instrukcje }

Opis:
Wykonuje się określoną liczbę razy, gdy znamy ilość powtórzeń.

Pętla while

Konstrukcja:
while (warunek) { instrukcje }

Opis:
Warunek sprawdzany przed wejściem – może się nie wykonać ani razu.

Pętla do while

Konstrukcja:
do { instrukcje } while (warunek);

Opis:
Warunek sprawdzany po wykonaniu – zawsze wykona się przynajmniej raz.
