Pytania teoretyczne

1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

Odpowiedź:

Właściwość danego obiektu można określić na dwa sposoby:

Sposób 1

nazwa_obiektu.nazwa_właściwości = "tekst";
auto.marka = "Opel Omega";

Sposób 2

nazwa_obiektu["nazwa_właściwości"] = wartość;
auto["cena"] = 25000;

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Odpowiedź:

Do właściwości i metod można się odwołać poprzez nazwę obiektu i kropkę.

console.log(samochod1.kolor); // wyświetla kolor obiektu samochod

3. Wymień cechy charakterystyczne konstruktora.

Odpowiedź:

  • Jest wywoływany automatycznie w chwili tworzenia obiektu.
  • Nazywa się tak samo jak klasa.
  • Nie zwraca żadnej wartości.
  • Może przyjmować argumenty.

4. Konstruowanie obiektu na podstawie przykładu – obiekt ma trzy pola oraz jedną metodę.

var osoba_1 = {
    nazwisko: "Kowalski",
    imie: "Jan",
    zawod: "piekarz",
    wyswietl: function () {
        document.write(this.nazwisko + " " + this.imie);
    }
};

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

Odpowiedź:

  • this – pozwala odwołać się do pola lub metody obiektu z wnętrza tego obiektu.
  • Pola i metody są oddzielane przecinkami.

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

function Klient(nazwisko_k, imie_k, zawod_k) {
    this.nazwisko = nazwisko_k;
    this.imie = imie_k;
    this.zawod = zawod_k;

    this.wypisz = function () {
        alert(this.nazwisko + " " + this.imie);
    };
}

7. Znaczenie słowa kluczowego New wraz z przykładem.

Odpowiedź:

Słowo kluczowe new służy do tworzenia nowego obiektu na podstawie konstruktora.

var osoba1 = new Klient("Kowalski", "Jan", "kierowca");
var osoba2 = new Klient("Nowak", "Anna", "sekretarka");

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

Odpowiedź:

Prototype pozwala na dodawanie nowych metod i właściwości do obiektów.

function Klient() {
    this.nazwisko = "Bielski";
    this.imie = "Paweł";
}

Klient.prototype.pisz_dane = function () {
    document.write(this.nazwisko + " " + this.imie);
};

Klient.prototype.zawod = "kierowca";

var osoba1 = new Klient();
osoba1.pisz_dane();

Opis: W definicji konstruktora nie zostały zadeklarowane metody i właściwości. Zostały one dodane później za pomocą prototype.
