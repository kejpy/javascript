Odpowiedzi teoretyczne do JavaScript

1) Kiedy powstał JavaScript (podaj datę).

JavaScript został stworzony w 1995 roku.

2) Dlaczego jest to język skryptowy?

JavaScript jest językiem skryptowym, ponieważ działa na poziomie przeglądarki i nie wymaga kompilacji do uruchomienia.

3) Gdzie jest wykonywany JavaScript (klient lub serwer)?

JavaScript jest wykonywany głównie po stronie klienta (w przeglądarce).

4) Dlaczego JavaScript to bezpieczne narzędzie?

JavaScript jest bezpieczny, ponieważ działa w izolowanym środowisku przeglądarki, co ogranicza jego dostęp do systemu operacyjnego użytkownika.

5) Czy jest to język obiektowy, jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ pozwala na tworzenie i manipulację obiektami.

6) Wymień zalety JavaScript:

Zaleta1: Łatwość integracji z HTML.

Zaleta2: Działa na większości przeglądarek bez konieczności instalowania dodatkowych wtyczek.

Zaleta3: Umożliwia tworzenie interaktywnych stron.

7) Framework to:

Framework to zestaw gotowych narzędzi i bibliotek wspierających tworzenie aplikacji.

8) Do aplikacji mobilnych można użyć Framework:

Tak, istnieją frameworki JavaScript, które wspierają tworzenie aplikacji mobilnych (np. React Native).

9) Aplikacje responsywne to:

Aplikacje, które automatycznie dostosowują się do rozmiaru ekranu urządzenia.

10) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu

‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›

Tworzy przycisk, który otwiera nowe okno przeglądarki po kliknięciu.

11) Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu

window.open("obraz.html", "okienko", "toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Otwiera nowe okno o określonych wymiarach i pozycji. Parametry `toolbar`, `directories` i `menubar` ukrywają elementy interfejsu.

12) Opisz następującą instrukcję window.close()

Instrukcja zamyka bieżące okno przeglądarki.
